# [1주차 학습 과제]
## 과제 : 로또 번호 추출기 

* Basic : 한세트의 로또번호 랜덤 샘플링 결과 출력
* Challenge : 사용자 입력 만큼의 로또 번호 세트 샘플링 결과 출력
